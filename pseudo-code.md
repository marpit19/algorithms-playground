## How to read pseudo code

- assigning to a variable: `index <- 1`
- `index <- size` reads the value of the variable size and assigns it to index
- Comparing Values:
    - `index == size`
    - `index <> size`

- for loop

```
n <- length(A)
total <- 0
for i in  (0..n-1) do
    total <- total + A[i]
```

- while loop

```
n <- length(A)
i <- 0
total <- 0
while i < n do
    total <- total + A[i]
    i <- i + 1
```
